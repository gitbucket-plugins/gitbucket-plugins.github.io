What is it about?

This is the community site for gitbucket plugins.

You will find here information and news regarding the gitbucket plugins maintained by the gitbucket community.

Community plugins list

  • Announce plugin

    Send announces via email to group of persons, allow to announce maintenance, upgrade, ...

  • H2 Backup plugin

    Introduce functionnalities to backup the H2 database used by gitbucket core.

  • Desktop notification

    Notify github activity directly on the desktop.

  • Gist plugin

    Add code snippet Gist like functionnality to gitbucket

  • Commits graph plugin

    Add commit graphs views to gitbucket

  • Asciidoc gitbucket plugin

    Add AsciiDoc rendering capabilities to gitbucket

  • Bugspots gitbucket plugin

    Add Google bugspots functionnality to gitbucket

  • Gitbucket Pages plugin

    Project pages for gitbucket

  • Gitbucket network plugin

    Network graph for gitbucket
