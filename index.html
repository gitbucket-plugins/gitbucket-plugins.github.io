This is the community site for all the public GitBucket plugins.

Available Plugins:

  • Announce plugin

    Send announcements via email to groups of users, e.g. for maintenance.

  • H2 Backup plugin

    Backup functionality of the H2 Database used by GitBucket.

  • Desktop Notification plugin

    Notify GitBucket activity directly on the desktop.

  • Gist plugin

    Add code snippet Gist-like functionality to GitBucket.

  • Commits Graph plugin

    Add commit graph views to GitBucket.

  • AsciiDoc plugin

    Add AsciiDoc rendering capabilities to GitBucket.

  • Bugspots plugin

    Add Google bugspots functionality to GitBucket.

  • Pages plugin

    Project Pages for GitBucket.

  • Network plugin

    Network graph for GitBucket.

  • Emoji plugin

    Emoji support for GitBucket.

  • RST plugin

    ReSTructured text renderer for GitBucket

  • Explorer plugin

    Explorer like navigation side panel for your projects.

  • FESS plugin

    Fast enterprise search for GitBucket using a FESS server.

  • PlantUML plugin

    Render PlantUML files in GitBucket.

  • Monitoring plugin

    Monitors and displays statistics about the running GitBucket instance.

  • HTML5 Media plugin

    Renders HTML5 Media (Audio/Video/PDF) inline in GitBucket.

  • Jupyter plugin

    GitBucket plugin for rendering Jupyter or IPython files.

  • Email Notifications plugin

    GitBucket plugin to send Email notifications.

  • CI plugin

    GitBucket plug-in that adds simple CI ability to GitBucket.

  • Maven Repository plugin

    GitBucket plugin that provides Maven repository hosting on GitBucket.

  • Mirror plugin

    This plugin adds repository mirroring to GitBucket.

  • H2Console plugin

    A GitBucket plugin that adds H2 console to the administration console.

  • Backup plugin

    Plugin providing backup functionality for GitBucket.

  • Logs Viewer plugin

    A plugin allowing to view and control GitBucket logs.

  • Kanban Board plugin

    A plugin for Kanban style issue management in GitBucket.

  • Gantt Chart plugin

    A plugin for rendering a Gantt Chart view of the GitBucket issues.

  • Swagger plugin

    A plugin for rendering a Swagger (OpenAPI) files.

  • R Markdown plugin

    A plugin for rendering a R Markdown files.

  • MathJax plugin

    A plugin for rendering a markdown files with MathJax.

  • 3D file plugin

    A plugin for rendering 3D files.

  • GeoJSON plugin

    A plugin for rendering GeoJSON files.

Your plugin is not listed here? You want to move your plugin under the organization? Please open a ticket/issue to reference your demand.
